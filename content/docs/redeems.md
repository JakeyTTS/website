If you stream on Twitch, you can link JakeyTTS to your Channel Points.

1. Create a Custom Reward on Twitch.
2. Link the exact Reward Name in JakeyTTS.
3. Configure whether the reward plays a specific voice, an SFX, or allows the user to type a message.
