# How to Create Plugins

Are you a developer looking to extend JakeyTTS? 

1. Learn the JakeyTTS API.
2. Create a new plugin folder using the standard template.
3. Write your custom logic.
4. Load your plugin into the `plugins` folder to test it locally.

*Detailed API references and starter templates will be provided here.*
